# AuraERP
AuraERP is an AI-powered Enterprise Resource Planning (ERP) web application designed for small and medium-sized businesses. It centralizes core business operations such as Inventory, Employee Management, Sales, Finance, and AI-powered document assistance into a single platform.

## Features

- 🔐 Secure JWT Authentication
- 📊 Interactive Dashboard
- 📦 Inventory Management
- 👥 Employee Management
- 💰 Sales & Finance Tracking
- 🤖 AI Assistant (Text-to-SQL)
- 📄 Document Manager with RAG
- 📈 Executive Report Generation
- 🐳 Dockerized Deployment

## Tech Stack

- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL
- **AI:** Google Gemini
- **Vector Database:** ChromaDB
- **Containerization:** Docker & Docker Compose
